### 求解代码

```java
public int minPathSum(int[][] matrix) {
        int m = matrix.length;
        int n = matrix[0].length;

        int[][] dp = new int[m][n];
        dp[0][0] = matrix[0][0];

        for (int i = 1; i < m; i++) {
            dp[i][0] = dp[i - 1][0] + matrix[i][0];
        }

        for (int j = 1; j < n; j++) {
            dp[0][j] = dp[0][j - 1] + matrix[0][j];
        }

        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                dp[i][j] = Math.min(dp[i - 1][j], dp[i][j - 1]) + matrix[i][j];
            }
        }
        return dp[m - 1][n - 1];
    }
```

### 小贴士

代码结构跟上一篇[【至下而上+动态规划】不同路径的数目（一）](https://blog.csdn.net/qq_44678890/article/details/157145928?spm=1001.2014.3001.5501)是一样的。

`dp[i][j]` 的定义：从起点 `(0,0)` 走到位置 `(i,j)` 的最小路径和。
