### 求解代码

```java
public int[] solve(int n, int m, int[] a) {
        m %= n;
        reverse(a, 0, n - 1);
        reverse(a, 0, m - 1);
        reverse(a, m, n - 1);
        return a;
    }

    private void reverse(int[] arr, int start, int end) {
        while (start < end) {
            swap(arr, start++, end--);
        }

    }

    private void swap(int[] arr, int i, int j) {
        int tmp = arr[i];
        arr[i] = arr[j];
        arr[j] = tmp;
    }
```

### 小贴士
因为`右移 n 位 = 没动`，`右移 n + k 位 = 右移 k 位`；

所以，要通过取余操作把 `m` 压缩到 `[0, n-1]` 范围。
