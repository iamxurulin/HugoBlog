### 求解代码

```java
public boolean judge(String str) {
        int n = str.length(); // 获取字符串长度，用于定义右指针初始位置
        // 双指针遍历
        for (int i = 0, j = n - 1; i < j; i++, j--) {
            // 逐位对比首尾字符，只要有一位不一致，直接判定不是回文
            if (str.charAt(i) != str.charAt(j)) {
                return false;
            }
        }
        // 所有对应位字符都一致，判定是回文
        return true;
    }
```
