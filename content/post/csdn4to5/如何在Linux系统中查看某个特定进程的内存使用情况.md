1.使用ps命令

举个例子，如果要查看进程ID为1527的进程的内存使用情况，可以使用如下命令：

```bash
ps -p 1527 -o %mem,%cpu,vsz,rss
```
其中，vsz表示虚拟内存的大小，rss表示常驻内存的大小。

2.使用top命令

要查看某个进程的内存使用情况时，可以依次输入top，再按f键进入域选择界面，再按N键就能显示内存的使用排序。

3.使用/proc文件系统

/proc提供了内核和进程的信息。

每个进程都有一个对应的目录，以ID为1527的进程为例，

/proc/1527/status包含VmSize虚拟内存大小和VmRSS常驻内存大小这些基本信息；

/proc/1527/statm包含进程的内存页面数。
