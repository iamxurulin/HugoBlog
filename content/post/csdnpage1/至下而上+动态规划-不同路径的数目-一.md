### 求解代码

```java
public int uniquePaths(int m, int n) {
        int[][] dp = new int[m][n];

        for (int i = 0; i < m; i++) {
            dp[i][0] = 1;
        }

        for (int j = 0; j < n; j++) {
            dp[0][j] = 1;
        }

        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                dp[i][j] = dp[i - 1][j] + dp[i][j - 1];
            }
        }
        return dp[m - 1][n - 1];
    }
```
### 小贴士

因为只能从上面或左边走过来，所以**递推公式**是
`dp[i][j]=dp[i-1][j]+dp[i][j-1]`；

同时，也不难得出第一行和第一列都是1的**边界条件**。
